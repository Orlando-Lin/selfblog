> Match the filename in `content/posts/` (e.g. `my-post.md` → `en/my-post.md`).

## Level-2 headings appear in the sidebar TOC

Write in **Markdown** — lists, quotes, tables (GFM), code blocks, and `inline code` are supported.

### Level-3 headings are included too

Run `npm run build` after saving; the post appears in the blog list when EN mode is on (or as fallback if no EN file exists).
