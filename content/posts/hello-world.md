时隔许久，我的个人博客终于完成了一次彻底的重写。这一次，我抛弃了原来的 Jekyll，转向了更现代的技术栈。

## 为什么要重做

老博客基于 Jekyll，虽然能用，但有几个让我难受的点：

- 主题定制起来不够灵活，想做点炫酷的动效很费劲；
- 本地开发体验一般，Ruby 环境配置略繁琐；
- 缺少组件化，复用和维护成本高。

于是我决定推倒重来，选一套自己更熟悉、也更面向未来的方案。

## 新的技术选型

| 方面 | 选择 |
| --- | --- |
| 框架 | Next.js 16（App Router） |
| 样式 | Tailwind CSS v4 |
| 动效 | Framer Motion |
| 内容 | Markdown + gray-matter |
| 部署 | GitHub Pages（静态导出） |

> 设计上走的是「毛玻璃 + 极简优雅」路线，配上渐变光晕背景和暗黑模式。
